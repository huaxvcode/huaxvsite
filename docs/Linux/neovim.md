# neovim

## 安装 neovim

```bash
sudo apt-get-repository ppa:neovim-ppa/stable
sudo apt-get update
sudo apt-get install -y neovim
```

## 普通模式下

### 上下左右

```
h       左

j       下

k       上

l       右
```

### 多行跳跃、多格跳跃

```
4h      向左边跳 4 格

4j      向下跳 4 行

4k      向上跳 4 行

4l      向右跳 4 格
```

### 单词跳跃

```
b       上一个单词，last beginning

w       下一个单词，next word
```
