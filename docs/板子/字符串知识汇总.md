## 字符串哈希

求字符串的哈希值，进制一般取 $233、13331、131$

```cpp
ull R = 233;
ull Hash(const std::string& s) {
    ull H = 0;
    for (auto c : s) {
        H = H * R + c;
    }
    return H;
}
```

---

获取字符串某区间的哈希值：

```cpp
const ll N = 2e6;
ull R = 233;
ull H[N], P[N];
std::string s; // 从 1 开始
ll n; // 字符串的长度
void init() {
    P[0] = 1;
    for (ll i = 1; i <= n; i ++) {
        H[i] = H[i - 1] * R + s[i];
        P[i] = P[i - 1] * R;
    } 
}
ull get(ll l, ll r) {
    return H[r] - H[l - 1] * P[r - l + 1];
}
```
