# 多类选择器

写法一：

```html
<style>
    .red.text { /*标签选择器*/
        color: red;
        font-weight: bolder;
    }
</style>
```

写法二：

```html
<style>
    .text.red { /*标签选择器*/
        color: red;
        font-weight: bolder;
    }
</style>
```

如果某个类含有多个类名，要怎么根据这个特征选上呢？用上述的多类选择器缩小范围

```html
<!DOCTYPE html>
<html lang="zh-CN">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body>
    
<!-- =========================================== -->
    <br>
        <span>
            <p class="red">hello world</p>
            <span> <p class="red text bord">hello python</p> </span>
        </span>
        
        <p>
            hello javascript
        </p>
        
        <em class="text">hello c++</em> </br>
        <span><em class="red">hello java</em></span>
    </div>
    <style>
        .red.text { /*标签选择器*/
            color: red;
            font-weight: bolder;
        }
    </style>
<!-- =========================================== -->
    
</body>

</html>
```
