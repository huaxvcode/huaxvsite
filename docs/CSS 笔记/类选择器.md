# 类选择器

```html
<style>
    .red, .text { 
        color: red;
        }
</style>
```

选中所有类名包含 `red` 和 `text` 的标签

```html
<!DOCTYPE html>
<html lang="zh-CN">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body>
    
<!-- =========================================== -->
    <br>
        <span>
            <p class="red">hello world</p>
            <span> <p>hello python</p> </span>
        </span>
        
        <p>
            hello javascript
        </p>
        
        <em class="text">hello c++</em> </br>
        <span><em class="red">hello java</em></span>
    </div>
    <style>
        .red, .text { 
            color: red;
        }
    </style>
<!-- =========================================== -->
    
</body>

</html>
```
