# id 选择器

```html
<style>
    #hw { 
        color: red;
        font-weight: bolder;
    }
</style>
```

选中 `id` 为 `hw` 的标签

```html
<!DOCTYPE html>
<html lang="zh-CN">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body>
    
<!-- =========================================== -->
    <br>
        <p id="hw"> hello world </p>
    </div>
    <style>
        #hw { 
            color: red;
            font-weight: bolder;
        }
    </style>
<!-- =========================================== -->
    
</body>

</html>
```
