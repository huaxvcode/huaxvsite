# 标签选择器

```html
<style>
    p, em { /*标签选择器*/
        color: red;
    }
</style>
```

将所有的标签 `<p>` 和 `<em>` 都选上

```html
<!DOCTYPE html>
<html lang="zh-CN">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>

<body>
    
<!-- =========================================== -->
    <br>
        <span>
            <p>hello world</p>
            <span> <p>hello python</p> </span>
        </span>
        
        <p>
            hello javascript
        </p>
        
        <em>hello c++</em> </br>
        <span><em>hello java</em></span>
    </div>
    <style>
        p, em { /*标签选择器*/
            color: red;
        }
    </style>
<!-- =========================================== -->
    
</body>

</html>
```
