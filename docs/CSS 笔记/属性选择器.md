# [属性选择器]()

```html
<style>
    [type="em"] { 
        font-style: italic;
    }
</style>
```

选中所有属性为 `text` 且属性值为 `em` 的标签

---

```html
<style>
    [text] {
        color: red;
    }
</style>
```

选中属性为 `text` 的所有标签

---

```html
<style>
    p[text="em"] {
        font-size: larger;
    }
</style>
```

选中所有属性为 `text` 且值为 `em` 的标签 `<p>`

---
